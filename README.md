# 项目简介 java167

## 项目概述

本项目是基于SpringBoot和Vue的高校就业管理系统，采用前后端分离的开发模式。系统涵盖了管理员、企业、辅导员、学生四种角色，实现了用户登录、注册以及就业相关的各项管理功能。

## 技术栈

- 开发工具：IDEA 2020.3，WebStorm 2020.3
- 运行环境：JDK 1.8，Maven 3.6.0，MySQL 5.7，Node.js
- 服务端技术：SpringBoot，MyBatis-Plus，Knife4j，Swagger-UI
- 前端技术：HTML，CSS，Vue，Axios，Element-UI

## 功能模块

系统主要功能包括：

- 用户登录
- 学生注册
- 商家注册
- 首页统计
- 个人信息管理
- 用户信息管理
- 学院专业管理
- 企业岗位管理
- 类型信息管理
- 简历信息管理
- 学生三方协议管理
- 就业统计

## 系统角色

- 管理员
- 企业
- 辅导员
- 学生

## 运行环境

- JDK 1.8 或更高版本
- Maven 3.6.0 或更高版本
- MySQL 5.7 或更高版本
- Node.js（适用于前端开发）

## 部署步骤

1. 创建数据库，导入SQL脚本
2. 修改`adminvue-base-web`模块下的`application.yml`中的数据库配置
3. 修改文件上传位置，设置图片路径
4. 启动服务端
5. 在前端项目目录下执行`npm install`或`yarn install`下载依赖
6. 依赖下载完成后，运行`npm run serve`启动前端服务
